# Framework Execution

There are two ways to execute the test cases

1) Using TestNG, you can execute from "TC_api_binance.java"
2) Using test runner class "TestRunner.java", which will execute BDD Cucumber feature

* Cucumber reports are generated in target folder only executed through TestRunner file

* Output print is also included

* Request URI is visible in request logs when testcases are executed

* Config.properties is used to store endpoint and base URLs
